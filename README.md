# EXA Billing 1.5 #

##

## How do I get set up? ##

### Install dependencies
* ``` npm run install-billing ```

### Debugging
* ``` npm run start ```

### Production deployment
* ``` npm run start-production ```

### Test
* ``` npm run test ```

### Build
* ``` npm run build ```

## Who do I talk to? ##

* Repo owner or admin
* Other community or team contact
